# P5-T13 — Automate the Homebrew Release Lifecycle

## Objective

Continuously validate the formula and supported macOS install lifecycle on Intel and Apple Silicon runners.

## Acceptance Criteria

- CI runs style/audit, clean source install, formula test, upgrade, uninstall, and config-preservation checks.
- Both supported macOS architectures are represented.
