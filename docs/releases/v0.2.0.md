# Tokenkeeper v0.2.2

Successor release for the post-`v0.1.0` hardening line. Earlier candidates are superseded by this immutable release.

Highlights:

- Native macOS ACL acquisition and conservative `acl_to_text` parsing.
- Complete anchored ancestor inspection, including replacement-capable ACL rights.
- Composable confidentiality/integrity policies for credential-bearing locations.
- Canonical Home handling for elevated invocation.
- Structured findings, unambiguous CLI scopes, platform-aware profiles, per-location evidence, and native acceptance fixtures.
- Pinned Rust 1.85.0 CI with locked coverage and Homebrew lifecycle workflows.

The existing `v0.1.0` tag is immutable and superseded; this release must identify the reviewed successor tree exactly.
